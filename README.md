# 🌍 Geo Quiz Bot (Географический КВИЗ-БОТ)
Интерактивный Telegram-бот для викторины по географии: страны, столицы, флаги; моря, реки, океаны и др.

## 🚀 Возможности

- 🏙 Угадай столицу страны
- 🌍 Угадай страну по столице
- 🚩 Викторина по флагам
- ♾  Бесконечный режим (до ошибок / на время)
- 📊 Подсчёт очков
- 🧠 Рандомные вопросы без повторов логики
- 🖼 Поддержка локальных изображений и URL-ссылок
